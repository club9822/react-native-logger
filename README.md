# react-native-logger

native logger

## Installation

```sh
npm install react-native-logger
```

## Usage

```js
import Logger from "react-native-logger";

// ...

const result = await Logger.multiply(3, 7);

const logger = Logger.logNativeError

logger('hello world : this log printed formated in logcat')

```
